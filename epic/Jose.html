Purpose: This project will be to create a professional user-friendly website for web developers. These developers will be able to create a profile to broadcast their skills and area of expertise. When recruiters go to the website they will be able to see the web developers skills which will make it easier to fill job postings.

Audience:The audience will be web developers and recruiters.

Goal: To help website developers to find jobs

Persona:

Situation: I am currently a technical recruiter who works 50 to 60 hour weeks. His stress level is high because it can be hard at time to find developers who fit the job requirements or has the needed skills for the job.

User Story As a recruiter, I am would like to have a website where I can see a brief snap-shot of a developers skills
