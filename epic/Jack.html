Purpose, Audience & Goal

Purpose:

To create a website where greenies and advanced web developers can setup profiles and search for job postings. It will also allow employers, recruiters and freelancers to post without setting up a profile.


Audience:

This site will be targeting professional individuals within web development and IT community, business owners, hiring managers, recruiters and web developers


Goal:

The primary goal of this site is for amateur to advanced skill based web developers to have a fast an easy way to find employment. It will also assist employers in posting jobs along with freelancers and recruiters contacting web developers.
